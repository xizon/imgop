# Image compression and optimization


### How to use?

```sh
$ cd ./scripts/imgop
$ npm run start
```

### You can modify the directory path of the file `img-op.mjs` to optimize wordpress by default

```js
...
const uploadsDir = path.join(__dirname, '../../../api/wp-content/uploads/2025');
...
```
